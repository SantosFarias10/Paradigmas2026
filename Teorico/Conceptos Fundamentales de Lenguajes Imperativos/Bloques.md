Construcciones estándar que estructuran los saltos. Como: 
* `if ... then ... else ... end`.
* `while ... do ... end`.
* `for ... { ... }`.
* `case ...`.
Agrupan el código en bloques lógicas, evitan saltos explícitos (excepto el retorno de la función).
No se puede saltar al medio de un bloque o función.

### Lenguajes con Estructura de Bloques
Permite tener bloques anidados con variables locales.
También nos ayuda en el manejo de memoria:
* Al entrar al bloque reservamos espacio para las variables.
* Al salir del bloque se puede liberar parte o todo el espacio.
